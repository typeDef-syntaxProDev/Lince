# LINCE

Landing page para marca de ropa deportiva femenina.

## Descripción

Este proyecto es una página web desarrollada con HTML, CSS y JavaScript
para una marca de ropa deportiva femenina.

Incluye:
- Página principal (Home)
- Página de detalle de producto
- Diseño responsive

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Objetivo

Practicar maquetado web, estructura profesional de archivos
y diseño moderno orientado a e-commerce.

## Mejoras futuras

- Implementar carrito de compras
- Conectar con backend
- Integrar pasarela de pago
- Agregar filtrado de productos
